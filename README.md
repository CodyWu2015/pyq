Quick install:

```
$ pip install --allow-external pyq pyq 
```

or

```
$ pip install -i https://pypi.enlnt.com -U pyq
```

For detailed installation instructions see [INSTALL](INSTALL.md).

Usage:

```
pyq [python options] python-script
```

Documentation:

```
pyq -m pydoc pyq
```

Test:

```
pyq-runtests
```
